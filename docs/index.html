Pretraining Recipes

This repository has been rebuilt as a Sphinx documentation project using the Sphinx Book Theme. The maintained docs source lives in docs/, with chapters under docs/source/.

Build locally:

cd docs
python -m pip install -r requirements.txt
make current-docs

The documentation covers VLA pretraining and mid-training stages, dataset families, recipe evidence, Cortex 2.0 deployment-data guidance, and quality gates for real, synthetic, and fleet telemetry.

If GitHub Pages is configured for Actions, pushes to main publish the generated Book Theme site automatically.
